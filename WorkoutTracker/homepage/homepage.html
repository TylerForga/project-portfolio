What does this website do?

This website allows you to learn new workouts and track your workouts and/or meals daily. You can search workouts by name or by which muscle group you want to workout under Workouts. The Macros and Workout Tracker allows you to add what you ate daily in order to track your macros. It allows you to add what workouts you did as well. Click on the Workouts or Macros & Workout Tracker tab at the top to get started.

*This is only a partially working demo of the website. Some functionalities will not work due to being hosted on Github and Github only serving static webpages.*

Future releases/updates for this website

Future releases and updates for this website include hosting on a different webhost other than Github. By obtaining a different webhost enables me to connect to a database and will allow me to integrate user accounts, allow searching for workouts as well as enable users to add their own workouts, and allow me to be able to save and retrieve your data you input into the Macro and Workout Tracker.

Functionalities currently unusable

  • The Macros and Workout Tracker will only save your data during the current visit to this website. Upon closing out of this website, your data in the calendar will be lost.
  • The Workouts page does not support searching for workouts by muscle group and is unavailable until I find a webhost that allows me to connect to a database.
  • The About page will not allow you to send me questions, comments or concerns until I find a webhost that allows me to connect to a database.
